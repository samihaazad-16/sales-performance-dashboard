# Sales Performance Dashboard

This project is an Excel dashboard built to analyze sales data by order type.

## Features
- Tracks sales by Delivery and Pickup
- Uses formulas such as SUMIF, AVERAGE, and COUNT
- Includes a bar chart to visualize sales distribution
- Organizes raw data into a structured dashboard format

## Tools Used
- Microsoft Excel
- Data analysis formulas
- Chart visualization

## Purpose
This dashboard demonstrates basic business analysis and reporting skills using Excel.
